## About The Project

### Overall
For Siege W12, I had originally tried to make an ad for framework using animation, but then deleted 4.5 hours of work in the editor and in blender. I did manage to figure out how to import stp and step files into blender to turn into gltf to give to Godot... All that is to say this project has 0 direction now since I lost motivation after deleting everything accidentally. 

### What even is this?
CRT Framework laptop thing? Render? Don't even know
(Siege W5)
* Nice animations
* Open and close laptop
* A and D to see expansion cards
* Hover over expansion cards to outline them and see a tooltip

### What do I take out of this?
Don't delete big files, always make backups, and learn blender. Also, don't animate in godot, it's terrible and you should just learn blender animation...

### Play 

If you'd like to build it yourself, I used Godot 4.5. I do have a pre-built web version available [eventually on github pages](https://pixelsaver.github.io/framework-horror), but if you still insist on building this unoptimized mess, go ahead.

1. Install Godot 4.5
2. Download and unzip the code
3. Open the file with Godot project manager
4. Go to Project > Export, add whichever platform you're on (MacOS, Windows) and then click export.
5. You're good to go!
